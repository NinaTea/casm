
# CASM - WIP

RV32I Assembly in Python just for fun. 

Lo que hace ahora:
- Agarra el assembly y printea su equivalente en binario.

objetivo cercano:
- Tomar un archivo en .asm y generar otro achivo que todas las lineas sean en binario
